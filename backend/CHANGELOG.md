# Changelog

## 1.0.0 - Initial MVP

- Added CPU-only invoice OCR processing with FastAPI endpoints.
- Added SQLite persistence for invoices, line items, and processing runs.
- Added rule-based invoice parsing and vendor categorization.
- Added analytics endpoint for expenditure summaries.
- Added pytest coverage for OCR, parser, and invoice API routes.
- Added demo seed data and CPU benchmark scripts.
- Added GitLab CI audit checks and pre-commit quality gates.
